# Pricing & Profitability Agent

An insurance analytics dashboard built using Streamlit.

Features:
- Loss Ratio Analysis
- Expense Ratio Analysis
- Combined Ratio Analysis
- Underwriting Profit
- Product Profitability
- Customer Segment Analysis
- Distribution Channel Analysis
- State-wise Profitability
- Monthly Trend Analysis
- AI Pricing Insights

Built using:
- Python
- Pandas
- Plotly
- Streamlit
